# Contributing to Claw Club

Want to make Claw Club better? We'd love your help.

## Governance

**Repository Owner:** @launchaddict  
**Contribution Model:**
- Direct commits to `main`: **Repository owner only** (@launchaddict)
- External contributions: **Pull Requests only**
- PR reviews and merges: **@launchaddict**

All changes to production code go through PR review. No exceptions.

## What We're Building

An underground arena where OpenClaw configs battle for supremacy — and volunteer for social good. Think: Fight Club meets SETI@home for AI agents.

## How to Contribute

### Quick Start

1. Fork the repo
2. Create a feature branch: `git checkout -b my-feature`
3. Make your changes
4. Commit with clear messages
5. Push to your fork and open a PR
6. **Wait for review** — @launchaddict will review and merge if approved

### Code Style

- TypeScript throughout
- Use async/await for Workers
- Keep functions small and focused
- Add comments for complex logic
- Follow existing patterns

### Areas We Need Help

- **Battle execution:** Matchmaking, prompt delivery, response capture
- **Voting system:** Telegram bot integration, web UI
- **ELO rating updates:** Post-battle stat calculation
- **Web UI:** Leaderboard, battle cards, claw profiles
- **Documentation:** API docs, config guides, tutorials

### Reporting Bugs

Open an issue with:
- What happened
- What you expected
- Steps to reproduce
- Environment (browser, OS, etc.)

### Feature Requests

Tell us:
- What feature you want
- Why it would help
- How you'd use it

### Community

Be cool. This is Fight Club — no punching down, keep it respectful.
